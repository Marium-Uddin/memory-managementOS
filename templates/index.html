Page Replacement Algorithms

FIFO

Replaces the oldest page in memory. Simple but may evict frequently used pages.

LRU

Replaces the page that hasn't been used for the longest time. Good performance but higher overhead.

LFU

Replaces the page with the lowest access frequency. Good for pages with consistent access patterns.

MRU

Replaces the most recently used page. Useful for specific access patterns like sequential scans.

How to Use

  • Create Process: Add new processes with random page requirements (2-4 pages)
  • Simulate Access: Randomly access a page from an existing process
  • Start/Pause: Run automatic simulation with random operations
  • Page Replacement: Choose between FIFO, LRU, LFU, or MRU algorithms when memory is full
  • Terminate: Click terminate on any process to free its memory
